# NASCON Database Management System

DBMS project developed for managing NASCON (National Student Convention), a large-scale event involving competitions, sponsorships, accommodations, and more.

This system automates complex event management processes and integrates multiple modules into one platform.

Key Features:
• Role-based user management (Admin, Organizer, Participant, Sponsor, Judge)
• Event creation & venue scheduling with conflict handling
• Sponsorship management & financial tracking
• Accommodation allocation system
• Judge evaluation & leaderboard generation
• Advanced reporting & analytics

Database Concepts Applied:
• Joins (INNER, LEFT)
• Aggregate functions (SUM, AVG, HAVING)
• Views, Indexes & Stored Procedures
• Triggers & Event Scheduler

Technologies Used:
MySQL, C#, SQL, Windows Forms, Visual Studio

Built a real-world database system with strong focus on normalization, constraints, automation, and efficient query design.
